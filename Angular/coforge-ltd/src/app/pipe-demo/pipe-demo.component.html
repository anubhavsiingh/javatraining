Angular Pipes

Pipes are referred as filters. It helps to transform data and manage data within interpolation, denoted by "|".

It accepts data, arrays, integers and strings as inputs which are separated by "|" symbol.

Today's date without pipe: {{presentDate}}

Today's date with pipe: {{presentDate | date}}

short date :- {{presentDate | date:'shortDate' }}
Full date :- {{presentDate | date:'fullDate' }}
Formatted date:- {{presentDate | date:'M/dd/yyyy'}}
Hours and minutes:- {{presentDate | date:'h:mm'}}

Built-in Pipes

Angular 8 supports the following built-in pipes.

AsyncPipe

If data comes in the form of observables, then Async pipe subscribes to an observable and returns the transmitted values.

Seconds changing in Time: {{ timeChange | async }}
Hours and minutes:- {{ timeChange | async| date:'h:mm'}}

The Async pipe performs subscription for time changing in every one seconds and returns the result whenever gets passed to it. Main advantage is that, we don't need to call subscribe on our timeChange and donâ€™t worry about unsubscribe, if the component is removed.

Currency Pipe

{{6589.23 | currency:"USD"}}
{{6589.23 | currency:"USD":true}} //Boolean true is used to get the sign of the currency.

Decimal Pipe

{{ 454.78787814 | number: '3.4-4' }} // 3 is for main integer, 4 -4 are for integers to be displayed.

Json Pipe

Percent Pipe

{{00.54565 | percent}}

Slice Pipe

{{months | slice:2:6}} // here 2 and 6 refers to the start and the end index
